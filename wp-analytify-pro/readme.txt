Welcome!

Analytify Pro works as an add-on of Core version which is Free and available at https://wordpress.org/plugins/wp-analytify/
You will install Core version first and then Pro version. Core and Pro versions are necessary to use other add-ons like WooCommerce, EDD, Automated Email notifications etc.

For more help, checkout our YouTube Channel videos and Subscribe at https://www.youtube.com/channel/UCRjDcNZqf-Vo_2Ux_QLyBCA/videos

Our Documentation is available at https://analytify.io/documentation/

Cheers,

Adnan from Analytify LLC.

== Installation ==

You should have installed Analytify Free + Pro versions to use this Analytify for WooCommerce add-on.

If you have any problem, please send an email to us at support@analytify.io
